## Description
A responsive single-page portfolio site for a junior developer, showcasing hero, about, projects, and contact sections using semantic HTML and either plain CSS or Tailwind CSS. Focus is on clean structure, good UI/UX, and mobile responsiveness.

## Tech Stack
- Bootstrap
- Tailwind CSS

## Requirements
- Create a hero section with developer name, title, and a clear CTA button.
- Implement an About Me section with a short introduction and image placeholder.
- Showcase 3 featured projects including titles, short descriptions, and GitHub links.
- Add a contact form with name, email, and message fields.
- Ensure the layout is mobile responsive.
- Add simple hover or scroll animations for visual interest.
- Deploy the site using GitHub Pages or Vercel and include the URL in the README.
